【ゲーム概要】このゲームはマスコットアプリ文化祭2015のキャラを使用したミニゲーム集です。
　　　　　　　現在、"ちぇんじ TO ずんだもち"と"カット パイナップル"の２本が収録されています。
　　　　　　　今後時間があればゲームを追加していく予定です。


【起動方法】マウスをダブルクリックか右クリック→"開く"を選択。
　　　　　　もし起動しない場合はご連絡ください。


【操作方法】ゲーム以外はキーボード操作です。
　　　　　　上下右左：矢印キー
　　　　　　ENTER,z　:決定
　　　　　　x　　　　:キャンセル、ひとつ前へ（ゲーム中は無効）


【インストール方法】ZIPファイルなので解凍したらすぐに遊べます。


【アンインストール方法】アプリが入っているフォルダごと削除して頂ければいいです。


【バージョンアップ方法】最新版はhttp://knct-gsaku.github.io/にあげます。
